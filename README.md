# 项目简介

## 核心功能
- 基于 YOLOv8 模型的淡水鱼检测系统，支持 30 类淡水鱼识别
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整的用户界面，可实时显示目标位置、目标总数和置信度
- 支持批量检测功能，可直接查看所有检测结果
- 可自定义检测参数：置信度阈值、IOU阈值
- 提供多种显示选项：可选择显示标签和原图
- 支持检测结果保存功能

## 适用场景
- 水产养殖研究与管理
- 水生生物多样性调查
- 淡水鱼种类识别与分类
- 水产资源监测与保护
- 水生生态系统研究

## 技术特点
- 采用 Python 3.9 开发环境
- 集成 OpenCV-Python、PyQt5、Ultralytics、Torch 1.9 等技术框架
- 提供 5500 张标注数据集，涵盖 30 种淡水鱼种类
- 包含完整源码、训练好的模型、数据集及项目运行教程
- 支持图形化操作界面，操作简便直观
